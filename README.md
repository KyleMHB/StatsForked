# Stats Forked

Stats Forked is a declarative GUI-table framework mod for the game RimWorld by Ludeon Studios. It includes several built-in tables, as well as compatibility patches for other mods.

This project is a fork of [AzzkiyOne/Stats](https://github.com/AzzkiyOne/Stats) and is maintained by kylohb at [KyleMHB/Stats-Forked](https://github.com/KyleMHB/Stats-Forked).
